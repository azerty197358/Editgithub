## 🚀 النشر على Render (دقيقة واحدة!)

### 1️⃣ انشر مباشرة:

اضغط الزر الأزرق → **Connect** → **Deploy!**

### 2️⃣ أو يدوياً:

1. افتح: **https://render.com**
2. اضغط **"New +"** → **"Blueprint"**
3. اربط مستودع GitHub
4. اضغط **"Apply"**
